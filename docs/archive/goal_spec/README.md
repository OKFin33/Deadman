# Archived Goal Specs

These files are historical execution contracts from the sprint.

They are retained as evidence of how Deadman artifacts were produced, reviewed,
and promoted. They are intentionally no longer in the main `docs/goal_spec/`
entry directory because many are superseded, noisy, or tied to the original
OSeria-Alter workspace shape.

Use archive files for:

- audit trail;
- reconstructing artifact provenance;
- understanding why later contracts exist;
- checking old task boundaries before rewriting history.

Do not use archive files for:

- current public setup instructions;
- current product claims;
- current v0.4 UX direction;
- proof that every planned feature shipped.

# Deadman Assets

Owns Branch 3 static assets.

`public/` is served by the standalone frontend through Vite `publicDir`, so
asset URLs resolve under paths such as
`/assets/branch3/companion/tomato-robes/...`.

Production builds may copy these assets into frontend build output. Those
copies are generated host artifacts; edit the source assets here.

Do not place raw MP4/MOV drama source files here.
